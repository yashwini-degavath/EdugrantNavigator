# Edugrant Navigator

Edugrant Navigator is a web-based platform that helps students explore educational grants and scholarship opportunities easily.

## Features

* Search educational grants
* User-friendly interface
* Responsive design
* Fast navigation

## Technologies Used

* HTML
* CSS
* JavaScript
* Node.js

## Project Structure

```text
EdugrantNavigator
├── backend
├── frontend
└── README.md
```

## How to Run

```bash
git clone https://github.com/yashwini-degavath/EdugrantNavigator.git
```

Open the project folder and run the application locally.
